## Scripts Included

### 1. System Identity Report (script1.sh)
Displays kernel, user, uptime, OS details.

### 2. Package Inspector (script2.sh) 
Checks Git installation and details.

### 3. Disk & Permission Auditor (script3.sh)
Analyzes system directories, permissions, disk usage.

### 4. Log File Analyzer (script4.sh)
Counts keyword occurrences in log files.

### 5. Manifesto Generator (script5.sh)
Creates personalized open-source manifesto.

## Setup & Run (Terminal Only)

1. Clone repo:
```bash
git clone https://github.com/manasvi010-debug/open-source-audit-project.git
cd open-source-audit-project
```

2. Make executable:
```bash
chmod +x *.sh
```

3. Run scripts:
```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh test.log error
./script5.sh
```

## Requirements
- Linux / Google Cloud Shell
- Bash shell
